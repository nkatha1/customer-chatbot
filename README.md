# Customer Support Chatbot

A simple Python-based chatbot to handle customer support tickets and FAQs.  
This chatbot allows users to create support tickets, view frequently asked questions, and check all submitted tickets.

---

## Features

- Create support tickets with unique IDs
- Suggest FAQs based on user input
- View all submitted tickets with status and timestamp
- Command-based interface for easy interaction

---

## Commands

- `help` - Show available commands  
- `faq` - Show FAQ questions  
- `ticket <issue description>` - Create a support ticket  
- `tickets` - Show all tickets  
- `exit` or `quit` - Close the chatbot  

Example:  
ticket My app crashes when I open it
